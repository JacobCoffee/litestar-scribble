Usage Examples¶

Basic Plugin Setup¶

from litestar import Litestar
from scribbl_py import ScribblPlugin, ScribblConfig

app = Litestar(
    plugins=[ScribblPlugin(ScribblConfig())],
)

Custom Configuration¶

from scribbl_py import (
    ScribblPlugin,
    ScribblConfig,
    InMemoryStorage,
)

config = ScribblConfig(
    storage=InMemoryStorage(),
    api_path="/api/v1",
    enable_websocket=True,
)

app = Litestar(plugins=[ScribblPlugin(config)])

Creating a Canvas Programmatically¶

from scribbl_py import Canvas, Stroke, Point, ElementStyle

# Create a canvas
canvas = Canvas(
    name="My Drawing",
    width=800,
    height=600,
)

# Create a stroke
stroke = Stroke(
    points=[Point(x=10, y=10), Point(x=100, y=100)],
    style=ElementStyle(stroke_color="#000000", stroke_width=2),
)

Module Overview¶

Plugin (scribbl_py.plugin)

The Litestar plugin and configuration classes for integrating scribbl-py into your application. See Plugin.

Core (scribbl_py.core)

Domain models including Canvas, Element, Stroke, Shape, Text, and Point. Also includes type enums and styling classes. See Core.

Storage (scribbl_py.storage)

Storage backends for persisting canvas data. Includes the StorageProtocol interface, InMemoryStorage, and optional DatabaseStorage. See Storage.

Services (scribbl_py.services)

Business logic services including CanvasService, ExportService, and TelemetryService. See Services.

Realtime (scribbl_py.realtime)

WebSocket handlers for real-time collaboration including the CanvasWebSocketHandler and ConnectionManager. See Realtime.

Auth (scribbl_py.auth)

OAuth authentication with support for Google, Discord, and GitHub. Includes User, Session, and UserStats models. See Auth.

Web (scribbl_py.web)

HTTP controllers and routes including CanvasController, ElementController, and the UIController for the web interface. See Web.

Game (scribbl_py.game)

Canvas Clash game logic including Room, Player, Round models and the word bank system. See Game.
