Factory Functions¶

scribbl_py.create_router(path='/api')[source]¶

Create the scribbl-py API router.

This function creates a Litestar Router with all the API controllers registered under the specified base path.

Parameters:

path¶ (str) – The base path for all API routes. Defaults to “/api”.

Return type:

Router

Returns:

A configured Litestar Router instance.

Example

>>> router = create_router("/api/v1")
>>> # Use the router in your Litestar app configuration
